Product: WSO2 IS outbound provisioning connector for Duo Security
Pre-requisites:

- Maven 3.x
- Java 1.7 or above

Tested Platform:

- Ubuntu 14.04
- WSO2 IS 5.1.0
- Java 1.7

Do the following:

Deploying and Configuring Tiqr artifacts:
1.  Build < DUO_PROVISIONING_COMNNECTOR_HOME>/component/authenticator & copy the org.wso2.carbon.identity.provisioning.connector.duo-1.0.0-SNAPSHOT.jar into <IS-HOME>/repository/components/dropins

2.  Follow the steps in https://docs.wso2.com/display/ISCONNECTORS/Configuring+Duo+Provisioning
